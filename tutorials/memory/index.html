Understanding Hermes Memory

Hermes' memory system is what sets it apart from other AI assistants. It's not just a simple chat history; it's a sophisticated system that allows the AI to remember and recall information across sessions, making it truly personalized and powerful.

1. The Memory Engine

At the core of Hermes is a memory engine that stores information in a structured way. It uses a combination of techniques to ensure that important information is retained while less important details fade away.

2. Memory Markdown (.memory.md)

The primary storage for Hermes' memory is a markdown file located at ~/.hermes/memory.md. This file contains all the facts, preferences, and contextual information that Hermes has learned about you.

You can view and edit this file directly to add or modify information. However, it's recommended to use Hermes' built-in memory commands to ensure proper formatting.

3. Peer Cards

For each person you interact with, Hermes creates a "peer card" that stores information about that individual, including their communication style, preferences, and past interactions. This allows Hermes to tailor its responses when communicating with or about specific people.

4. Fuzzy Index and Prompt Cache

Hermes uses a fuzzy index to quickly retrieve relevant memories based on semantic similarity, not just exact keyword matches. Additionally, a 1-hour prompt cache helps reduce token usage by storing recent context.

5. Adding Your Soul (soul.md)

One of the most powerful features is the ability to create a soul.md file. This file acts as a context manual for your life, telling Hermes who you are, where you live, your goals, preferences, and how you want it to behave.

By providing this information, Hermes can better understand your context and provide more personalized assistance.

6. Connecting External Systems

Hermes can connect to various external systems to enhance its memory:

  • Obsidian: Link your Hermes memory to your Obsidian vault for seamless knowledge sharing.
  • Email: Connect to your email (via services like Zapier) to allow Hermes to recall and act on email content.
  • Meeting Notetakers: Integrate with tools like Granola, Fireflies, or Fathom to recall meeting discussions.

7. Practical Tips for Enhancing Memory

  • Regularly review and update your soul.md file.
  • Use specific, detailed prompts when teaching Hermes new information.
  • Leverage the peer card system by mentioning people's names in conversations.
  • Connect external systems gradually to avoid overwhelming the memory system.
  • Use the hermes memory commands to manage your memory directly.
