Model Switching & Delegation

Learn how to switch between different AI models and delegate tasks to specialized models for optimal performance.

1. Model Switching

Hermes allows you to switch between different AI models on the fly, depending on the task at hand. You can use the /model command to see available models and switch to the one best suited for your current task.

For example:

  • Use Opus 4.7 for writing and creative tasks
  • Use Grok for real-time data and social media analysis
  • Use specialized models for coding, math, or reasoning tasks

2. Delegation to Specialized Models

Beyond just switching models, you can delegate specific tasks to different models. This is particularly useful for complex workflows where different steps require different strengths.

Example workflow:

  • Use a research model to gather information
  • Use a writing model to draft an article
  • Use a coding model to implement a solution
  • Use a review model to proofread and improve

3. Anti-Gravity CLI (Gemini Alternative)

With the deprecation of the Gemini CLI, Hermes now supports the anti-gravity CLI, which provides access to Gemini models for multimodal tasks like image and video analysis.

You can use the anti-gravity CLI within Hermes to:

  • Analyze images and videos
  • Process multimodal inputs
  • Leverage Gemini's advanced reasoning capabilities

4. Practical Applications

  • Content Creation: Use different models for research, writing, editing, and optimization.
  • Problem Solving: Delegate different aspects of a problem to models specialized in math, logic, or creativity.
  • Learning: Use models that excel at explanation for learning new concepts.
  • Development: Use coding models for development tasks and other models for documentation or design.

5. Best Practices

  • Match the model to the task: use the right tool for the job.
  • Experiment with different models to find what works best for each type of task.
  • Consider token costs and response times when selecting models.
  • Use delegation to break down complex tasks into manageable steps.
  • Stay updated on new models and capabilities as they become available.
